# Additional Additions
This mod allows you to use new additions for [Legend of Dragoon: Severed Chains](https://legendofdragoon.org/projects/severed-chains) created using the [Additional Additions Editor](https://github.com/Legend-of-Dragoon-Modding/Additional-Additions-Editor).

## How to Install
Download the mod (.jar) from the releases page of this repository and put it in your Severed Chains `mods` folder.

## How to Use
1. Install the mod (see previous section for installation instructions)
2. Copy the `.json` files for the additions you want to use into the `mods/additional-additions` directory in your Severed Chains directory
   1. If this directory doesn't exist, you can create it
3. You will be able to use your new additions in any campaign that has the Additional Additions mod enabled
   1. To add Additional Additions to an existing campaign, highlight your campaign in the campaign selection screen and press `Mods` at the bottom of the screen

## Creating New Additions
You will need the [Additional Additions Editor](https://github.com/Legend-of-Dragoon-Modding/Additional-Additions-Editor) to make new additions.
